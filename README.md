# The Forge

A personal fitness and nutrition tracker built for two users, hosted on GitHub Pages.

Track workouts, log meals, and monitor progress — all from a lightweight static web app with no backend required.

Records stay in each browser's local storage. Use **Settings > Export** inside the app to create a portable JSON backup, and restore it before moving to a different browser or device. The installed app shell works offline after the first successful load.

**Live site:** https://trailerbrosaz.github.io/The-Forge/
